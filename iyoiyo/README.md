# IYOIYO 个人助理站

## 一键部署
1. 将本项目解压到任意目录。
2. 双击 `start.bat` 启动 PHP 内置服务器（需先安装 PHP 或使用内置 PHP）。
3. 启动后访问 `http://localhost:19266` 即可使用。
4. 首次启动后建议先访问 `http://localhost:19266/iyo/iyo.php` 查看相关配置与说明。

## 最高权限快捷配置（！重要！不使用时请关闭）
- 访问`http://localhost:19266/iyo/iyo.php`
- 上述地址为默认路径，可以手动修改文件名，修改后需填写新文件名才可以访问。（更多信息可以访问该文件查看）
- 默认密码为 `123456`（如果忘记，请查阅iyo文件夹下的 `admin_config.txt` 文件）

## 功能
- 账号独立认证（JSON文件）
- 搜索引擎集合（百度/Bing/...）
- 收藏夹管理（分类+书签）
- 我的订阅（大分类→标签→条目，支持封面、进度、备注）
- 我的手札（分类+笔记）
- 我的提醒（一次性/循环/永久）
- 个人配置（头像/背景/密码/主题）
- 手机/PC自适应
- ai接入（如需本地部署，可自行安装ollama搭建）

## 目录结构
详见项目文档。

## 注意事项
- 确保 `data` + `log` + `public/uploads` 目录可写。
- 如需外网访问，可使用 `cloudflared` 等工具穿透，配合动态DNS更新域名。

## 重要文件说明
- data/
	- `iyoiyo.db` 数据库文件，删除即清空所有信息
- iyo/ （文件夹重命名后，请同步更新config.json）
	- `iyo.php` 快捷权限入口文件
	- `admin_config.txt` iyo平台管理员明文密码文档（勿丢失、流出）
- `config.json` 平台配置文件
- `users.json` 平台内所有用户必要信息（勿丢失、流出）

- log/ （运行日志）
	- `web.log` php内置服务器的log记录
- `tunnel.log` 内网穿透获取地址自动生成的文件
- `ai_debug.log` ai对话日志


- bak_tools/ （额外独立备用脚本，如需使用，请将其移动到上一级目录运行）
	- `start.bat` cmd方式的总启动
	- `start_web.bat` 站点地址（执行顺序：第1顺位，运行后的cmd界面不可以关闭，关闭就会使这个程序不再运行）
	- `start_tunnel.bat` 内网穿透（执行顺序：第2顺位，运行后的cmd界面不可以关闭，关闭就会使这个程序不再运行）
	- `cleanup.bat` 清理进程、端口占用
